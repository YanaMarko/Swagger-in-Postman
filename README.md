# Учебный проект: Тестирование REST API (Postman & Swagger)

учебный проект для изучения основ тестирования API, работы с технической документацией и автоматизацией.

## 🎯 Цели проекта
* Изучить структуру REST API и форматы передачи данных (JSON)..
* Освоить отправку запросов и валидацию ответов в Postman.
* Написать первые автоматические тесты.

## 🛠 Технологический стек & Инструменты
* **Swagger UI ** — для анализа документации и ручной проверки эндпоинтов.
* **Postman** — для создания коллекций, настройки окружения и автоматизации тестов.
* **JavaScript (Postman Sandbox)** — для написания скриптов валидации (тест-раннер `pm.test`).
* **JSON** — в качестве формата тела запросов (Request Body) и ответов (Response Body).

## 🛠 Что протестировано
* **Позитивные сценарии:** Создание заказа, поиск заказа по id, удаление заказа, поиск животного по статусу, добавление животного и т.д.
* **Негативные сценарии:** Поиск животного по id после его удаения, поиск заказа по id после его удаления

## 🚀 Как запустить тесты локально
1. Установите десктопное приложение **Postman**.
2. Скачайте содержимое этого репозитория.
3. В Postman нажмите кнопку **Import** и выберите скачанный файл.
